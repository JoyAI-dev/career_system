# Career Exploration Platform — Documentation

A structured career cognition growth platform for university students. The system drives career development through three mechanisms: Cognitive Boundary Questionnaires, Community Learning Activities, and Post-Activity Cognitive Updates — forming a closed-loop growth cycle.
